# 命令行使CPU满载
很多时候, 我们想测试一下电脑的散热情况和风扇转速, 但是又不想下载其他软件来整个全负载, 其实, 命令也可以使cpu满载的.
1. 打开【终端】,输入 `yes > /dev/null &` 即可!
2. 这条命令只能是1个CPU满载, 如果电脑是多核, 可以多次执行该命令
3. 打开【资源管理器】或者【活动监视器】查看CPU运行状态
4. **最重要的:** 在【终端】使用 `killall yes`
