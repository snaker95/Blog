# 第六天 记录
## 1. calendar 模块
* `isleap(year)` : 判断year是否为闰年, 返回boolen
* `leapdays(y1, y2)` : 在y1 到 y2这一区间内有N闰年, 返回整型
* `weekday(year, month, day)`: 判断当前日是周几, 返回整型(0-6=>'日=>六')
