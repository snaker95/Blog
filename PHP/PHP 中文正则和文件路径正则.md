# PHP 中文正则和文件路径正则
在数据验证中, 匹配常见中文, 路径匹配
<!--more-->
## 一. PHP (中文)
php中utf-8编码下用正则表达式匹配汉字的最终正确表达式

```text
/^[\x{4e00}-\x{9fa5}]+$/u
```

## 二. JS (中文)
JS中正则表达式匹配汉字的最终正确表达式

```text
/^[\u4e00-\u9fa5]+$/
```

## 三. PHP、JS (文件路径)

```js
/^(([A-Za-z]:[\\\|\/])|\/)([^\\\\?\/\*\|<>:"]+[\\\|\/])*([^\\\\?\/\*\|<>:"]+\.[^.\\\\?\/\*\|<>:"]+)*$/
```

**讲解:**
> 文件的所在磁盘  :  (([A-Za-z]:[\\\|\/])|\/)
> 文件的基本路径  :  ([^\\\\?\/\*\|<>:"]+[\\\|\/])*
> 文件名称及后缀  :  ([^\\\\?\/\*\|<>:"]+\.[^.\\\\?\/\*\|<>:"]+)*$/
